====================
 pyais CHANGELOG
====================
-------------------------------------------------------------------------------
 Version 1.4.0  6 Mar 2021
-------------------------------------------------------------------------------


* Pyais comes with a commandline utility now
    - `ais-decode` can decode single messages or files
    - you can read from a TCP/UDP socket from the terminal
    - no need to write Python code

* Improves Socket API
